Particle Simulation: Parallelization with OpenMP, MPI, and CUDA

This project showcases three distinct implementations of a basic particle simulation using C++. Each implementations employs a different parallelization technique: shared memory parallelization using OpenMP, distributed memory parallelization using MPI, and GPU parallelization using CUDA.

All three versions in this project were developed and tested to run on Perlmutter, a supercomputer at The National Energy Research Scientific Computing Center (NERSC).


GitHub repo available upon request.

Cellular Automata Simulation

In this project, I worked in a cross functional team to examin the spread of a virus through a population of cells using a C++ cellular automata model.

This project features an organized, well-documented repository to test and run the model. The program architecture revolves around three main components: the cellular automata framework itself, the individual cells comprising the automata, and the simulation engine responsible for executing the experiment. Additionally, we utilized Matplotlib and Seaborn to create an animation of our cellular automata simulation.

Quantum Chemistry Algorithms

This project centers around the Complete Neglect of Differential Overlap/2 (CNDO/2) method, a semi-empirical quantum mechanical used to estimate molecular electronic structure and potential energy surface, using C++.

My program features a large, complex, and organized class to fascillate the subroutines associated with the CNDO/2 algorithm. In order to improve the overall performance, I utilized shared memory parellelization using OpenMP to improve the computational efficiency of key regions of my program. This allows my program to handle relatively large molecules.

GitHub

Cell Failure Mode

In collaboration with Sepion Technologies, a company centered around developing lithium metal and ion battery components for electronic vehicles, this project aimed to completely automate the failure detection process in battery prototype testing. Traditionally, this process involved manual examination of extensive time series datasets or resource-intensive structural teardowns, leading to inefficiencies and potential oversight of critical failures.

To address these challenges, I worked in a cross functional team of two to developed a custom Python based anomaly detection algorithm tailored to Sepion's specific testing processes. The final product consisted of a comprehensive web application and well-documented Python package.
